搭建个人博客步骤

搭建个人博客原本是"云计算"课程的一门小作业, 刚好借这个机会把个人博客给搭建好. 根据费曼学习法, 也督促我把所学的东西写写博客, 锻炼自己各方面的能力. 另外, 趁着刚把个人博客搭建好, 把搭建的步骤详细记录一下.

听起来云计算和个人博客似乎关联性不大?
逻辑链条是这样的, 云计算 -> DevOps -> 运维 -> 持续集成 -> Github Pages -> 个人博客
总的来说, 搭建个人博客的框架网上以及提供了无数现成的方案了, 自己要做的其实并不多, 过程也是出乎意料地简单
